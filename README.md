4. Technologies Used
Backend: PHP, Laravel

Database: MySQL
Development Tools: ChatGPT for code assistance
5. Key Features
Multi-Level Marketing (MLM) functionality
Investment tracking
User management
OTP verification
User authentication
Dashboard for users
Admin panel for managing investments and users
Secure transactions
Responsive design
Real-time notifications
